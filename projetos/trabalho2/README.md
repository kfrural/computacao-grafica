# Trabalho 2 - Balada

Aplicacao em Three.js para apresentar uma cena de balada com:

- luz ambiente branca;
- luz direcional amarela;
- tres luzes coloridas vermelha, verde e azul em movimento com efeito de pisca;
- dois objetos centrais usando `MeshLambertMaterial` e `MeshPhongMaterial`.

## Como executar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
